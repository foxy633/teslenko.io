Строчные теги :

  • a - создает ссылку на URL: веб-страницу, раздел на странице, адрес электронной почты ... Также называется anchorэлементом, откуда aприходит
  • b - делает элемент жирным
  • br - определяет разрыв строки в тексте
  • button - определяет нажимаемую кнопку
  • code - определяет фрагмент кода внутри блока текста
  • em - определяет акцент на тексте, обычно отображаеться как курсив
  • i - делает элемент курсивом
  • img - изображение
  • s - определяет зачеркнутый текст
  • strong - определяет большое значения для текста
  • u - подчеркивает текст

Блочные теги :

  • address - определяетблок для контактной информации
  • article - определяет автономный блок контента, который может существовать в любом контексте. Он может иметь собственный заголовок, нижний колонтитул, разделы ... Полезно для списка сообщений в блоге
  • blockquote - определяетдлинную цитату
  • body - контейнер для содержимого веб-страницы
  • footer - определяет нижний колонтитул веб-страницы или раздела
  • h1...h6 - заголовки
  • head - определяет контейнер для метаданных веб-страницы
  • header - определяет заголовок веб-страницы или раздела
  • nav - определяет раздел с навигационными ссылками
  • ol - определяет упорядоченный список
  • p - обзац текста
  • section - определяет раздел на веб-странице
  • ul - oпределяет неупорядоченный список
  • html - oпределяет корневой элемент документа HTML. Все остальные элементы должны содержаться в этом корневом элементе
